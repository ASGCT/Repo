# Uninstall-Webroot

This Script will uninstall webroot from a target machine. 

## Syntax
```PowerShell
Uninstall-Webroot.ps1 [<CommonParameters>]
```
## Description

Uninstalls Webroot

## Examples


###  Example 1 
```PowerShell
Uninstall-Webroot.ps1
```

Uninstalls Webroot from a target machine.
